## IPFS_gui - class project 2022
This app is incomplete and under development.  

## Requirements
Linux (tested on Arch linux)
Python 3.10.2

## How to run this app?
Clone this project.  
Run app.py with python:  
    python app.py

## issue

    sysctl -w net.core.rmem_max=2500000
